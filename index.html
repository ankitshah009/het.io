Heterogeneous Networks for bioinformatics

Heterogeneous (information) networks are graphs with multiple node and edge types. They provide a scalable, intuitive, and frictionless data structure for intergrating diverse information domains.

Heterogeneous Network Link Prediction

The goal of HNLP is to produce biologically-meaningful predictions by integrating multiple high-throughput data sources. The approach computes features describing the network topology connecting two nodes. These features are used as input to a machine learning method which predicts the probability that an edge exists. Through evaluating the informativeness of each feature, the relevance of included domains can be compared providing insight into the underlying forces driving the process of interest.

Disease-Associated Genes

Using heterogeneous network link prediction on a network with 22 node types (metanodes) and 24 edge types (metaedges), we predict the probability that each protein-coding gene is associated with each of 23 complex human diseases. Reported associations from the GWAS Catalog were used to train the model. Potential applications include prioritizing GWAS analyses, determining the causal gene within a genomic region, identifying genes of biological interest, and comparing the ability of various information domains to identify pathogenic variants.
